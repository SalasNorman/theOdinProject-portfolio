# Tic Tac Toe

A simple Ruby terminal implementation of the Tic Tac Toe game.

## Description

This project provides a command-line Tic Tac Toe game supporting a human player vs another human or the computer. It includes board display, move validation, win/draw detection, and a basic AI opponent.

## Features

- Two-player (human vs human) and human vs computer modes
- Clear terminal board display
- Detects wins and draws
- Validates player input

## Requirements

- Ruby (version 2.0 or higher)

## Usage

1. Clone or download the repository.
2. Navigate to the `tic_tac_toe` directory.
3. Run the game:

```bash
ruby main.rb
```

Follow the on-screen prompts to play.
